# pdd-183 网站访问流量看板

## 简介
网站实时访问数据可视化看板，展示在线人数、访问量趋势、访客来源、热门页面排行及设备分布。

## 技术栈
- React 18 + TypeScript + Vite
- TailwindCSS
- ECharts (echarts-for-react)
- lucide-react

## 安装与运行

```bash
npm install
npm run dev
```

打开浏览器访问 `http://localhost:3001`。

## 功能
- 顶部数字卡片：当前在线人数、今日总访问量、平均停留时长、跳出率
- 访问量趋势折线图：支持小时视图与日视图切换
- 访客来源饼图：搜索引擎、直接访问、社交媒体、外部链接、邮件营销
- 热门页面排行列表：页面路径、访问次数、平均停留时间
- 设备分布环形图：桌面端 vs 移动端
- 数据每 5 秒自动刷新模拟实时效果
